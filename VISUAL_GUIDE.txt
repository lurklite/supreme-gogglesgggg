║                 GITHUB PAGES SETUP - VISUAL GUIDE              ║
╚════════════════════════════════════════════════════════════════╝

┌─────────────────────────────────────────────────────────────────┐
│  STEP 1: Find Your GitHub Pages URL                            │
└─────────────────────────────────────────────────────────────────┘

    GitHub Repository → Settings → Pages
    
    Look for: "Your site is published at:"
    
    ┌──────────────────────────────────────────────┐
    │ https://lurklite.github.io/supreme-goggles   │  ← COPY THIS
    └──────────────────────────────────────────────┘


┌─────────────────────────────────────────────────────────────────┐
│  STEP 2: Add URL to Discord Developer Portal                   │
└─────────────────────────────────────────────────────────────────┘

    discord.com/developers/applications → Your App → OAuth2
    
    Scroll to "Redirects" section:
    
    ┌──────────────────────────────────────────────────┐
    │ Redirects                                        │
    │                                                  │
    │ ┌──────────────────────────────────────────────┐ │
    │ │ https://lurklite.github.io/supreme-goggles   │ │ [X]
    │ └──────────────────────────────────────────────┘ │
    │                                                  │
    │ ┌──────────────────────────────────────────────┐ │
    │ │ http://localhost:8080                        │ │ [X]
    │ └──────────────────────────────────────────────┘ │
    │                                                  │
    │ [+ Add Another]                                  │
    │                                                  │
    │                    [Save Changes] ← CLICK THIS!  │
    └──────────────────────────────────────────────────┘


┌─────────────────────────────────────────────────────────────────┐
│  STEP 3: Edit config.js on GitHub                              │
└─────────────────────────────────────────────────────────────────┘

    Your Repository → Click "config.js" → Click Pencil Icon
    
    BEFORE:
    ┌──────────────────────────────────────────────────┐
    │ CLIENT_ID: 'YOUR_DISCORD_CLIENT_ID',             │
    │ CLIENT_SECRET: 'YOUR_DISCORD_CLIENT_SECRET',     │
    └──────────────────────────────────────────────────┘
    
    AFTER:
    ┌──────────────────────────────────────────────────┐
    │ CLIENT_ID: '1234567890123456789',                │
    │ CLIENT_SECRET: 'aBcDeF123456789xyz',             │
    └──────────────────────────────────────────────────┘
    
    Then: Commit changes → Wait 1-2 minutes


┌─────────────────────────────────────────────────────────────────┐
│  WHAT YOU'LL SEE WHEN IT WORKS                                 │
└─────────────────────────────────────────────────────────────────┘

    1. Visit: https://lurklite.github.io/supreme-goggles
    
    2. See login screen with green debug box:
       ┌───────────────────────────────────────────────┐
       │  🔧 Debug Info                                │
       │  Redirect URI: https://lurklite.github.io/... │
       │  Add this EXACT URL to Discord Portal         │
       └───────────────────────────────────────────────┘
    
    3. Click "Login with Discord"
    
    4. Discord asks: "Authorize LURKOUT Script Builder?"
    
    5. Click "Authorize"
    
    6. Redirected back → You're logged in! ✓


╔════════════════════════════════════════════════════════════════╗
║                    COMMON MISTAKES                             ║
╚════════════════════════════════════════════════════════════════╝

    ❌ WRONG                         ✅ CORRECT
    ───────────────────────────────────────────────────
    http://lurklite.github.io...     https://lurklite.github.io...
    (must be https)                  (with s)
    
    .../supreme-goggles/             .../supreme-goggles
    (no trailing slash)              (no slash at end)
    
    Didn't click "Save Changes"      Clicked "Save Changes"
    (changes not saved!)             (saved properly)
    
    config.js not updated            config.js has real IDs
    (still has placeholders)         (actual credentials)


╔════════════════════════════════════════════════════════════════╗
║                    TROUBLESHOOTING                             ║
╚════════════════════════════════════════════════════════════════╝

    PROBLEM: "Invalid Redirect URI"
    ├─ Check: Green debug box shows your URL
    ├─ Check: That exact URL is in Discord Portal
    ├─ Check: You clicked "Save Changes" in Discord
    └─ Try: Wait 5 minutes, clear cache, try again
    
    PROBLEM: Changes not showing up
    ├─ Wait 1-2 minutes for GitHub to deploy
    ├─ Hard refresh: Ctrl+Shift+R (Win) or Cmd+Shift+R (Mac)
    └─ Try: Incognito/Private mode
    
    PROBLEM: "Failed to fetch"
    ├─ This is normal! Discord blocks browser OAuth token exchange
    ├─ App automatically falls back to Demo Mode
    └─ All features work perfectly in Demo Mode!
